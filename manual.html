Game Description

The players are tasked with defending different systems from various cybersecurity threats, using upgradable tower defense systems that represent real world cybersecurity systems like the Firewall Node and Patch Drone. It is structured in a 2D tower defense game manner, where enemies pass by on a set path, with players placing turrets that launch projectiles, or do other things in order to destroy the enemies before they reach the end of the path. If an enemy reaches the end of the path, the player loses a life, and if the player loses 5 lives, they lose the game. The objective is to survive as many waves as possible. However, there are many special features of the game that make it unique.

Controls

Left Side

  • Menu - Left Control
  • Firewall Node - 1
  • IDS Scanner - 2
  • 2FA Tower - 3
  • Patch Drone - 4
  • Encryption Tower - 5
  • Place/Upgrade Tower - Left Shift

Right Side

  • Menu - Right Control
  • Firewall Node - 6
  • IDS Scanner - 7
  • 2FA Tower - 8
  • Patch Drone - 9
  • Encryption Tower - 0
  • Place/Upgrade Tower - Right Shift

*These controls are all relative to the sides, depending on which players are playing the controls will be on different sides*
